dmu
============================

##Description
一个简单的豆瓣音乐客户端和last.fm scrobber (__Under_Developing__)

##Features
    1. 收听豆瓣电台
    2. 任务栏控制菜单
    3. Growl提醒
    4. 全局热键播放
    5. 模拟iTunes发布歌曲信息
